Development Roadmap & Token System Evolution

DeSocX Platform Development Phases

The DeSocX platform follows a structured development roadmap where different token types and features are introduced in sequential phases, each building upon the previous:

1

Pre-Launch: Treasure Hunt & Initial User Acquisition

Key Token Components:
  • Feynman Diagram Certificates: Entry NFTs that can be minted and later converted to Particle-Energy
  • Particle-Energy Tokens: Created from "collapsed" FDC NFTs with unique properties
  • Anyon-Shell: Biometric identity token created through entanglement
User Activity Flow:
User participates in cryptic treasure hunt
Receives airdrop with access credentials
Mints Feynman Diagram Certificate NFT
Creates Anyon-Shell via EyeDKey biometric system
Entangles Particle-Energy with Anyon-Shell
Features Unlocked:
  • Initial platform access
  • Basic dashboard features
  • Countdown to Day of Emergence
  • Preparation for $ANYN token launch
2

Day of Emergence: Token Launch

Key Token Components:
  • $ANYN: The reserve currency token officially launches
  • $sANYN: Staked ANYN token becomes available
  • Quintessence Boosts: Pre-launch participants receive enhanced rewards
User Activity Flow:
Purchase $ANYN tokens during 7-day auction
Stake $ANYN to receive $sANYN
Benefit from boosted yields based on pre-launch participation
Participate in the Red Queen Race
Features Unlocked:
  • Full platform functionality
  • Staking rewards based on quintessence tier
  • Basic "Earn and Save" loyalty benefits
  • Initial marketplace access
